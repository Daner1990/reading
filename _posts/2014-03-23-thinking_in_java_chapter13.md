###本章主题

在Web应用中，字符串算是使用率最高的结构了。我们查看JDK文档会发现，**String对象是不变的**，String类中每一个看起来会修改String对象的方法，实际上都是创建了一个全新的String对象，以包含修改后的字符串内容。而最初的String对象则丝毫未动。

在字符串领域有3个大头：

1. String
2. StringBuffer
3. StringBuilder
